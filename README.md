- Vite + React
- Sass

## Installation

To transfer the project to yourself, follow these steps:

1. Clone the repository:

```
git clone https://github.com/your-username/Brokol-Web.git .
```

2. Install the required dependencies:

```
npm install
```

3. Run the dev version:

```
npm run dev
```
